# Flappy Bird Marrouche
Un jeu flappy bird crée avec unity. Ceci est mon premier jeu sur unity. En suivant un tutoriel sur youtube qui m'a beaucoup aidé avec comment unity marche et comment créer tous les différentes choses qu'on peut créer, j'ai pu créer un jeu plutôt fonctionnel.

Je devais créer l'oiseau qui avait une force gravitationnel sur lui et a chaque fois qu'on appui sur espace, il saute. En réalité, l'oiseau ne bouge pas, c'est tout ce qui est autour qui bouge. Je devais ensuite créer les tuyaux, un qui viens vers le haut et un vers le bas et ensuite les positionner de façon aléatoire qu'il y a un trou quelque part sur l'écran. Puis, avec les tuyaux créée, je devais créer un générateur de tuyaux pour que ca soit à l'infini.
A la fin, j'ai ajouté quelques effets en plus comme du son et aussi des points a gauche. 

N'hesitez pas de jouer au jeu ( avec le .exe ) ou revoir le code en clonant le répertoire.
